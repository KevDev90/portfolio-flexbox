Fantastic Food Finder

Your one stop application for rating and leaving reviews about Restaurants you've visited! This project was built for people who would like a personal way of keeping track of restaurants they have visited in the past! I created this project for my final fullstack capstone at Thinkful!

Powder and Chowder

This project is an app that allows users to select a particular colorado ski resort on a map. After selecting a resort, that resorts weather data will be shown on the screen. This project was built for ski and snowboard users in the Colorado area. I created this project as my first API capstone project at Thinkful!

Tech stack used: HTML/CSS, JQuery, the Fetch API, Yelp Api, Weatherbit API, and Leaflet.
